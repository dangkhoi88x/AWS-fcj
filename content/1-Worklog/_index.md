**On this page**, you will need to introduce your worklog. **How** did you complete it? How many weeks did you take to complete the program? **What** did you do in those weeks?

I completed the AWS Cloud Journey program over **12 weeks** (from September 29, 2025 to December 21, 2025), going through a learning process from basics to advanced AWS services and implementing a real-world project. Below is an overview of the completed weeks:

**Week 1 (Sep 29 - Oct 5):** [Getting familiar with AWS and basic AWS services](1.1-week1/)
- Getting familiar with AWS cloud computing platform, core service groups (Compute, Storage, Networking, Database)
- Hands-on practice with AWS Free Tier, AWS Console, and AWS CLI
- Understanding basic concepts and initial operations with Amazon EC2

**Week 2 (Oct 6 - Oct 12):** [AWS IAM, EC2 deployment, VPC networking, and fundamentals](1.2-week2/)
- Learning and applying AWS IAM to manage identities and access
- Hands-on practice deploying and managing EC2 Instances
- Exploring Amazon VPC and AWS Site-to-Site VPN to understand basic networking concepts
- Reinforcing foundational knowledge of C# programming language

**Week 3 (Oct 13 - Oct 19):** [High Availability, Scalability, RDS, Aurora, ElastiCache, Route 53](1.3-week3/)
- Understanding core concepts of High Availability and Scalability in cloud environments
- Learning about AWS managed database services such as RDS, Aurora, and ElastiCache
- Learning how to configure and manage Amazon Route 53 for DNS and traffic routing
- Researching typical AWS solution architecture patterns

**Week 4 (Oct 20 - Oct 26):** [Amazon S3 storage, CloudFront, and integration services](1.4-week4/)
- Learning in detail about Amazon S3, CloudFront, Global Accelerator, Storage Gateway, FSx
- Learning how to use integration services such as SQS, SNS, Step Functions
- Hands-on practice designing and deploying storage and content delivery solutions

**Week 5 (Oct 27 - Nov 2):** [Containers on AWS, Serverless Architecture, and Database services](1.5-week5/)
- Learning how to deploy and manage containers on AWS using ECS, EKS, and Docker
- Understanding concepts and benefits of Serverless architecture
- Designing and building Serverless applications integrating Lambda, API Gateway, S3, and DynamoDB
- Learning about AWS database services such as RDS, Aurora, DynamoDB, Redshift, and ElastiCache

**Week 6 (Nov 3 - Nov 9):** [Data & Analytics, Machine Learning, and Monitoring](1.6-week6/)
- Exploring Data & Analytics services such as Athena, Glue, Kinesis
- Learning about Machine Learning with SageMaker and Rekognition
- Learning how to use CloudWatch and CloudTrail to monitor and track systems
- Advancing knowledge of IAM and access management

**Week 7 (Nov 10 - Nov 16):** [AWS Security & Encryption, VPC, and Disaster Recovery](1.7-week7/)
- Understanding and applying data and system security methods in AWS
- Managing, configuring, and optimizing Amazon VPC for private network systems
- Learning how to backup, restore, and implement disaster recovery plans
- Learning about security services such as KMS, CloudHSM, Shield, WAF

**Week 8 (Nov 17 - Nov 23):** [Advanced Solutions Architecture, IoT, and AWS Whitepapers](1.8-week8/)
- Researching advanced solution architectures on AWS
- Learning about IoT Core, Ground Station, RoboMaker
- Reading and analyzing AWS Whitepapers to master best practices
- Designing complex solutions with multiple integrated AWS services

**Week 9 (Nov 24 - Nov 30):** [Final AWS Cloud Journey summary report and system architecture design](1.9-week9/)
- Completing the summary report of the entire AWS Cloud Journey
- Building and presenting a complete system architecture based on learned AWS services
- Reviewing and consolidating all knowledge from Week 1 → Week 8
- Preparing reports, documentation, and final presentation content

**Week 10 (Dec 1 - Dec 7):** [Real-world deployment of Mini-Market system on AWS](1.10-week10/)
- Real-world deployment of the Mini-Market system on AWS environment
- Completing code and integrating designed AWS services
- Performing testing and optimizing system performance
- Preparing demo and video presentation of the system's main features

**Week 11 (Dec 8 - Dec 14):** [Final review and presentation preparation](1.11-week11/)
- Final review of all documents, reports, and proposal
- Completing presentation slides and presentation content
- Practicing presentation and preparing answers for Q&A
- Ensuring quality of all deliverables before submission

**Week 12 (Dec 15 - Dec 21):** [Final Presentation and program summary](1.12-week12/)
- Conducting Final Presentation on the entire AWS Cloud Journey and Mini-Market project
- Presenting and demonstrating the deployed system on AWS
- Summarizing the entire program and evaluating achieved results
- Receiving feedback from mentors and AWS community
