### Week 6 Goals

- Explore and practice **Data & Analytics** services on AWS.
- Gain hands-on experience with **Machine Learning (ML)** services such as **SageMaker** and **Rekognition**.
- Improve skills in **monitoring, auditing, and performance optimization** using CloudWatch, CloudTrail, and Trusted Advisor.
- Learn about **Advanced Identity and Access Management** on AWS, including IAM Roles, Federated Access, and SSO.

### Week 6 Achievements

- Developed a solid understanding of **AWS Data & Analytics Services**:

  - Queried data directly from S3 using **Athena**.
  - Built an ETL pipeline using **AWS Glue**.
  - Set up real-time data streams using **Kinesis** and explored **EMR** for big data workloads.

- Gained hands-on experience in **Machine Learning on AWS**:

  - Trained and deployed a simple ML model with **SageMaker**.
  - Used **Rekognition** for image and facial recognition.
  - Explored additional ML services such as **Comprehend**, **Translate**, and **Textract**.

- Strengthened skills in **Monitoring, Auditing, and Performance Optimization**:

  - Monitored metrics and logs with **CloudWatch**.
  - Tracked API activity with **CloudTrail** for better visibility and governance.
  - Used **Trusted Advisor** for performance and cost optimization recommendations.

- Improved understanding of **Advanced Identity Management**:

  - Configured advanced **IAM Roles** with least privilege access.
  - Implemented **Federated Access** for secure third-party authentication.
  - Set up **AWS SSO** for simplified multi-account access control.

- Successfully combined knowledge of **Data, Machine Learning, Monitoring**, and **Security**, building a strong foundation for next week’s topics on **Automation & Infrastructure as Code (IaC)**.
