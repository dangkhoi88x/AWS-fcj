### Key Highlights

#### Security Context and Challenges

- **Escalating cyber threats**: DDoS attacks increasing year-over-year and application-layer attacks targeting critical business functions.
- **Breach costs**: Organizations without robust perimeter protection face average breach costs of $4.88M and potential business disruption.
- **Importance of Perimeter Protection**: Need for proven AWS security frameworks to stay ahead of emerging threats.

#### From Edge to Origin: CloudFront as Your Foundation

**Amazon CloudFront - Global CDN**:

- **Edge Locations**: Understanding how CloudFront uses global edge location network to deliver content closer to users.
- **Core CDN Capabilities**: Learning core CDN capabilities from edge locations to security controls.
- **Security Controls**: Integrating security measures into CloudFront distribution.
- **Architecture & Optimization**: How to architect and optimize global content delivery.
- **Scalability**: Building robust foundation for serving content at scale.

**Audience**: Cloud architects and developers.

#### Attack Surface Defense: WAF & Application Protection

**AWS WAF - Web Application Firewall**:

- **Block Malicious Traffic**: Using AWS WAF to block malicious traffic before it reaches applications.
- **Mitigate OWASP Top 10**: Protecting against common OWASP Top 10 vulnerabilities like SQL injection, XSS, CSRF.
- **Defend Against Bot Attacks**: Detecting and blocking automated bot attacks.
- **Custom Rules**: Creating custom rules to protect specific applications.
- **CloudFront Integration**: Integrating WAF with CloudFront for edge protection.

#### Hands-On Workshop: Optimize Internet Web Application

- **CloudFront Practice**: Configuring CloudFront distribution for web applications.
- **Optimization Techniques**: Learning performance optimization techniques and latency reduction.
- **Caching Strategies**: Setting up effective caching strategies.
- **Real-world Scenarios**: Discussing real-world scenarios and implementations.
- **Best Practices**: Applying best practices from AWS experts.

#### Hands-On Workshop: Secure Internet Web Application

- **AWS WAF Practice**: Configuring AWS WAF rules to protect applications.
- **Security Configuration**: Setting up security measures for web applications.
- **Attack Simulation**: Experiencing how WAF blocks real attacks.
- **Monitoring & Logging**: Setting up monitoring and logging for security events.
- **Integration Patterns**: Learning how to integrate WAF with other AWS services.
